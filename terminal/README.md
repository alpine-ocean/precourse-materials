# TERMINAL

## QUESTIONS

- What are the top 20 commands for mac Command Line Interface (CLI)?

        1. cd: change directory
        2. ls: list contents of directory
        3. open: opens file
        4. cp: copy to another directory
        5. mv: move file
        6. touch: create new text file
        7. mkdir: make/create new directory
        8. rmdir: remove empty directory
        9. rm -R: remove a directory and all of it's contents
        10. sudo: super user do - super user permissions
        11. top: stats of system, similar to task manager
        12. q: quit
        13. clear: clear terminal 
        14. ditto: copy a directory and all of its contents into another directory
        15. whatis: provides a short description of a command
        16. exit: kills a terminal session
        17. vim: open text editor
        18. man: open manual for command
        19. ls -la: list directory contents including hidden . files
        20. Ctrl + c: quit current process

- What is a terminal? A CLI? Why are they synonymous?

    Historically speaking the terminal was a physical object that someone would go to that was the point of interface to the mainframe computer located in a different physical location - before personal computers. 

    The CLI - OR - Command line interface is where we communicate with the shell program that interfaces with the computer/kernel. 

    So the contemporary terminal is the CLI or window that we use to communicate with a computer.

- What is the difference between bash and zsh?

    bash and zsh are two different shell programs that are used to communicate with the kernel. bash is ore universal, whereas zsh has additional features and reverse compatible with bash.

- What is the difference between Terminal, Console, Shell, and Command Line?

    the terminal OR console hosts the shell program that you communicate with via the CLI to access the kernel.
