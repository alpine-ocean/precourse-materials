+ What are the top 20 commands for mac Command Line Interface (CLI)?

1. cd: change directory
2. ls: listing directory
3. open: opens files
4. cp: copy to another directory
5. mv: move file
6. touch: create a new text file
7. mkdir: make directory
8. rmdir: remove empty directory
9. rm -R: remove a directory and all of it's children
10. sudo: super user permission to execute command
11. top: stats of system, similar to task manager
12. q: quit
13. clear: clear terminal 
14. ditto: copy a folder and all of its children to another folder
15. whatis: provides a short description of a command
16. man: displays the manual for specified command
17. exit: kills the terminal session 
18. (apple only) shortcuts run
19. (apple only) tmutil - time machine back up
20. (apple only) killall "AppName"

+ What is a terminal? A CLI? Why are they synonymous?

A terminal is a text based window with a program running in it called  a shell (BASH, ZSH etc.), CLI stands for Command Line Interface (vs a GUI Graphical User Interface) that you use to interact with the shell running in the terminal. Terminal and CLI are often treated as synonymous because they are presented as one package.

+ What is the difference between bash and zsh?

both are shell programs, zsh is bash + more features ( zsh is backwards compatible with bash but has more features)
